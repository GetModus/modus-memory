# Homing CLI

`cmd/homing` is the primary binary entrypoint for **Homing by MODUS** as of `v0.6.0`.

The compatibility alias remains available under [`cmd/modus-memory`](../modus-memory/README.md) for existing scripts, MCP client configurations, and wrapper flows that still expect the legacy command name.
